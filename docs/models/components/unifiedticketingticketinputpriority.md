# UnifiedTicketingTicketInputPriority

The priority of the ticket. Authorized values are HIGH, MEDIUM or LOW.

| `UnifiedTicketingTicketInputPriorityHigh`   | HIGH                                        |
| `UnifiedTicketingTicketInputPriorityMedium` | MEDIUM                                      |
